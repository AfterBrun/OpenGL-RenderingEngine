## OpenGL_Rendering_Engine


### 1. 프로젝트 소개
* 본 프로젝트는 OpenGL을 이용하여 작성된 Renderer 입니다.

### 2. 적용 기술
1. Blinn Phong Shading
2. Normal Map
3. Shadow Map
4. Height Map을 이용한 지형 렌더링
5. Assimp를 활용한 모델 불러오기

### 3. 사용 라이브러리
* OpenGL 3.3: 3D 렌더링
* GLFW 3.x: 윈도우 생성 및 이벤트 처리
* Assimp: obj, fbx등 모델 파일 로딩
* stb_image: image파일 로딩
* spdlog: 콘솔 log출력
* GLM: 선형수학 라이브러리
* ImGui: Ui 렌더링
